## Run the project 

  to run the BE 

  open Keystone.sln file in BE folder with Visual studio

  ---------------------------------------------------------
  to run the FE 

 1- Navigate to the FE directory 

 2- run "npm install" to install the node modules 

 3- run " ng serve " to build and run FE
